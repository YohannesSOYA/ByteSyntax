# Tech Stack - Backend

- **Language**: Python 3.10+
- **Framework**: FastAPI
- **ASGI Server**: Uvicorn
- **Database**: 
  - ORM: SQLAlchemy
  - Migration: Alembic
  - Engine: PostgreSQL (Recommended) or SQLite
- **Validation**: Pydantic v2
- **Authentication**: OAuth2 with JWT (Jose)
- **Environment**: Pydantic-settings
